# Surprisal as the Drive of Subjective Time

This repository contains the manuscript and supporting materials for:

**Surprisal as the Drive of Subjective Time: A Gauge-Constrained Taxonomy and Its Decisive Test**
Nathan Hangen

## Overview

A companion formal note ([Gauge-Invariant Time Perception](https://github.com/nhangen/gauge-invariant-time-perception)) proves that any reparameterization-invariant readout of the proper-time trajectory alone collapses to an endpoint functional — so interior sensitivity to subjective time requires a path-distinguishing, τ-intrinsic *drive*. That result is silent on *which* drive.

This paper proposes a principled candidate: the **surprisal** of experience, `s = −log p`, so that felt duration is accumulated information, `ΔP = k ∫ s dτ` — gauge-invariant by construction. From this single currency it derives a battery of falsifiable predictions (a finite-ceiling age curve, an order effect in which clustered events read longest, a clean separation from contextual-change accounts, and Weber–Fechner scaling), tests them against digitized lifespan data and the open Blursday database, and is explicit about where the data already strain the account. It closes with a single decisive behavioural contrast that separates surprisal from its nearest rival.
